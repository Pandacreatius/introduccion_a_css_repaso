1. Estructura de una regla CSS

Una regla CSS siempre combina un selector con una o varias declaraciones. Cada declaración está formada por una propiedad y un valor.

Partes principales

Haz clic para resaltar cada parte de la regla.

  • Selector: indica a qué elemento se aplicará el estilo.
  • Propiedad: indica qué característica queremos cambiar.
  • Valor: indica cómo queremos cambiar esa propiedad.
  • Declaración: es la pareja formada por propiedad y valor.
Idea clave: selector + propiedades + valores.

Vista visual

h1 {
  color: #2c3e50;
  font-size: 2rem;
  margin-bottom: 20px;
}
SelectorElemento al que se aplica el estilo.
DeclaraciónPropiedad + valor, siempre con punto y coma.
PropiedadLo que queremos modificar.
ValorEl resultado que asignamos a la propiedad.

        Medidas principales
        
3. Unidades de medida en CSS: px, %, vw y vh

        
Estas unidades permiten definir tamaños, anchos, altos, separaciones y otros valores visuales.

      


      

        

          
Compara unidades

          
Cambia la unidad y observa cómo se interpreta el tamaño.

          
    
            
  • px: unidad fija. Muy útil para bordes, sombras, iconos y detalles pequeños.
    • 
            
  • %: unidad relativa al contenedor padre. Útil para anchos fluidos.
    • 
            
  • vw: 1vw equivale al 1% del ancho de la ventana.
    • 
            
  • vh: 1vh equivale al 1% del alto de la ventana.
    • 
          

          
px mantiene un tamaño fijo.

        Unidades relativas
        
4. em, rem y otras unidades útiles

        
em y rem son muy útiles para tipografía y espaciados proporcionales. También existen unidades como fr, ch, cm, mm, in, pt y pc.

      


      

        

          
Prueba em y rem

          
Ambas son relativas al tamaño de fuente, pero no toman la misma referencia.

          
    
            
  • em: depende del tamaño de fuente del elemento padre.
    • 
            
  • rem: depende del tamaño de fuente del elemento raíz: html.
    • 
            
  • fr: reparte espacio en CSS Grid.
    • 
            
  • ch: se basa en el ancho aproximado del carácter “0”.
    • 
          

          
em escala según el tamaño del padre.

        Efectos y organización
        
6. Texto y fuentes II

        
Estas propiedades añaden personalidad al texto, pero deben usarse con moderación para no perjudicar la lectura.

      


      

        

          
Prueba efectos

          
Aplica propiedades avanzadas y comprueba cuándo pueden ser útiles.

          
    
            
  • text-shadow: añade sombra al texto.
    • 
            
  • white-space: controla espacios y saltos de línea.
    • 
            
  • word-spacing: modifica el espacio entre palabras.
    • 
            
  • text-indent: aplica sangría a la primera línea.
    • 
            
  • text-overflow: controla texto que no cabe.
    • 
            
  • vertical-align: alinea elementos en línea respecto al texto.
    • 
          

          
No abuses de los efectos: úsalos para guiar la atención.

        Resumen final
        
7. Qué debe recordar el alumno

        
Ideas clave para repasar antes de practicar con ejercicios de HTML y CSS.

      


      

        
Regla CSS
Combina selector, llaves, propiedades, valores y punto y coma.
h1 { color: blue; }

        
Selectores
Sirven para elegir qué elementos reciben los estilos.
.clase · #id · etiqueta

        
Unidades
Elige la unidad según si quieres tamaño fijo, fluido o relativo.
px · % · vw · vh · em · rem

        
Texto
Debe ser legible, jerárquico y coherente en toda la página.
font-size · line-height · color

      


      

        
Buena práctica 1Usa nombres de clases claros y descriptivos, como .tarjeta, .menu o .aviso.

        
Buena práctica 2Cuida el contraste de color: el texto debe poder leerse sin esfuerzo.

        
Buena práctica 3Ordena tu CSS por bloques y deja sangría para facilitar la lectura.
